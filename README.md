# Predicción del Nivel de PIB con Datos del Banco Mundial

El objetivo de este proyecto es predecir el nivel de Producto Interno Bruto (PIB), medido en paridad de poder adquisitivo (PPP), utilizando indicadores económicos, sociales y demográficos obtenidos desde el Banco Mundial.

El desarrollo del proyecto se estructura en tres etapas. En primer lugar, se realiza un análisis descriptivo del conjunto de datos, junto con la identificación e imputación de valores faltantes y la discretización de la variable objetivo. En segundo lugar, se aplica Análisis de Componentes Principales (PCA) con el fin de reducir la dimensionalidad del conjunto de variables explicativas. Finalmente, se construyen modelos de clasificación para predecir el nivel de PIB de los países considerados.

Este enfoque permite analizar patrones estructurales asociados al nivel de desarrollo económico y evaluar el desempeño de distintas metodologías de modelación.
